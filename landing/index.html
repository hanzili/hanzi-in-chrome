Build with Hanzi

Add browser automation to your product.

Pair your users' browsers and run tasks from your backend. Your users see your app — Hanzi is the runtime underneath.

1 Sign in to your developer console
2 Send your user a pairing link — they click it and connect
3 Run a task via the REST API or TypeScript SDK
4 Show results in your app 
import { HanziClient } from '@hanzi/browser-agent';

const client = new HanziClient({ apiKey: 'hic_...' });
const result = await client.runTask({
  browserSessionId: session.id,
  task: 'Read the patient chart on this page',
});
console.log(result.answer);
